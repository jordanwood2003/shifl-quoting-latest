## What Shifl Quoting Does

Shifl Quoting is built for trucking quote creation and rate estimation in a web environment. It gives users a straightforward way to turn shipment information into a working estimate, reducing the friction involved in pricing tasks.

This project suits teams that want a browser-first quoting flow. By keeping trucking pricing and estimate generation in one place, it supports a consistent process that can be repeated across day-to-day operations.
